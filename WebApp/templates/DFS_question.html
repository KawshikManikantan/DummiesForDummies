Array with Queries

You are given an array A consisting of N positive numbers. You have to answer Q queries. Each query consists of an integer X, you have to print the length of smallest subarray of A starting from index 0 such that the sum of the numbers in the subarray is greater than or equal to X. If such subarray doesn't exist then print "-1"

Constraints

1≤N,Q≤106

1≤A[i]≤109

1≤X≤1018

Input

First line consists of 2 space seperated integers N and Q. The next line consists of N space seperated integers A[i]. The next Q lines contains the value of X where the ith line conatins value of X in ith query.

Output

Print Q space seperated integers, ith integer denoting the length of subarray in ith query or -1 if doesn't exist.

Sample Input

 6 4
 1 3 2 4 5 2
 6
 9
 20
 16

Sample Output

 3 4 -1 6
