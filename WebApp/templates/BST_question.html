Binary Search Tree

Try to implement insert function for BST and print the pre-order traversal of the created BST. You are given an integer N denoting the number of numbers you are going to insert.

Constraints

1≤N≤105

1≤Xi≤106 where Xi denotes the ith number inserted.

It is guaranteed that Xi≠Xj for i≠j

Input

First line of input contains N, the number of elements going to be inserted. The next line contains N space seperated integers where ith number denotes Xi

Output

Print N space seperated integers, the pre-order traversal of the BST.

Sample inputs

 3
 2 3 1
 4
 2 3 4 1
 2 1 3
 2 1 3 4
