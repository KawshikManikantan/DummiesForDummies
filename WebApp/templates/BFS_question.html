CoronaVirus

Recently, a new virus broke out in k cities in a country at time t=0. Country can be viewed as a (n x m) grid((0, 0) at the top left and (n-1, m-1) at the bottom right). In every second, the virus spreads from the infected cities to its neighbours that is the cells directly connected to it(up, right, bottom,left). Let time[i][j] denote the time at which the city i,j got the infection. Find the value of (∑((i + 1) x (j + 1) x time[i][j])) modulo 1e9 + 7

Constraints

1≤n,m≤1000

1≤k;≤n x m

Input

First Line contains n, m, k the number of rows, columns in the grid and the number cities which got infected at t=0. The next k lines contains 2 integers i,j denoting the cell infected

Output

Print an integer denoting the hash value

Sample Input

 3 3 1
 1 1
 2 2 1
 0 0

Sample Output

 48
 12
